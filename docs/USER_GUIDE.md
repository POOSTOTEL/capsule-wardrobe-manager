# Руководство пользователя

## Введение

Добро пожаловать в "Капсульный Гардероб" - веб-приложение для управления вашим гардеробом, создания образов и планирования капсульных коллекций.

## Начало работы

### Регистрация

1. Откройте приложение в браузере
2. Нажмите кнопку "Регистрация" или перейдите по ссылке `/register`
3. Заполните форму:
   - **Email** - ваш email адрес (будет использоваться для входа)
   - **Имя пользователя** - уникальное имя (минимум 3 символа)
   - **Полное имя** - ваше имя (опционально)
   - **Пароль** - минимум 6 символов
   - **Подтверждение пароля** - повторите пароль
4. Нажмите "Зарегистрироваться"

После успешной регистрации вы автоматически войдете в систему.

### Вход в систему

1. Перейдите на страницу входа (`/login`)
2. Введите ваш email и пароль
3. Нажмите "Войти"

Если вы забыли пароль, обратитесь к администратору (в текущей версии восстановление пароля не реализовано).

### Выход из системы

Нажмите кнопку "Выход" в правом верхнем углу страницы.

## Работа с вещами

### Добавление вещи

1. Перейдите в раздел "Вещи" через меню навигации
2. Нажмите кнопку "Добавить вещь"
3. Заполните форму:
   - **Название** (обязательно) - название вещи, например "Синяя куртка"
   - **Категория** (обязательно) - выберите категорию из списка:
     - Верх
     - Низ
     - Платье/Костюм
     - Обувь
     - Верхняя одежда
     - Аксессуар
     - Белье
   - **Фотография** (обязательно) - загрузите фото вещи
     - Поддерживаемые форматы: JPEG, PNG, GIF, WebP
     - Максимальный размер: 5MB
     - Изображение автоматически сжимается, если превышает 2MB
   - **Цвет** (опционально) - выберите цвет из справочника
   - **Сезон** (опционально) - выберите сезон:
     - Лето
     - Зима
     - Демисезон
     - Всесезон
   - **Теги** (опционально) - выберите один или несколько тегов
     - Можно создать новый тег прямо в форме
   - **Заметки** (опционально) - дополнительная информация о вещи
4. Нажмите "Сохранить вещь"

### Просмотр вещей

На странице "Вещи" отображается сетка всех ваших вещей с фотографиями. Каждая карточка показывает:
- Фотографию вещи
- Название
- Категорию
- Цвет (с визуальным индикатором)
- Теги (с цветовыми метками)
- Счетчик использования

### Фильтрация и поиск вещей

Используйте панель фильтров для поиска нужных вещей:

1. **Поиск по названию** - введите часть названия в поле поиска
2. **Фильтр по категории** - выберите категорию из списка
3. **Фильтр по цвету** - выберите цвет
4. **Фильтр по сезону** - выберите сезон
5. **Фильтр по тегам** - выберите один или несколько тегов
   - При выборе нескольких тегов показываются вещи, имеющие ВСЕ выбранные теги
6. **Сортировка** - выберите способ сортировки:
   - По дате добавления (новые сначала)
   - По названию (алфавитный порядок)
   - По использованию (чаще используемые сначала)

Все фильтры можно комбинировать. Для сброса всех фильтров нажмите кнопку "Сбросить".

### Просмотр деталей вещи

Кликните на карточку вещи, чтобы открыть страницу с детальной информацией:
- Фотография вещи
- Название
- Категория
- Цвет (с визуальным индикатором)
- Сезон
- Теги (с цветовыми метками)
- Заметки
- Статистика использования (сколько раз использовалась в образах и капсулах)
- Даты создания и обновления

### Редактирование вещи

1. Откройте страницу деталей вещи
2. Нажмите кнопку "Редактировать"
3. Измените нужные поля
4. Для замены изображения кликните на текущее изображение и выберите новое
5. Нажмите "Сохранить изменения"

**Примечание:** Изображение можно заменить, но это необязательно - если новое изображение не загружено, старое останется.

### Удаление вещи

1. На странице списка вещей найдите нужную вещь
2. Нажмите кнопку с иконкой корзины на карточке вещи
3. Подтвердите удаление в диалоговом окне

**Важно:** 
- Удаление необратимо
- Вещь будет удалена из всех образов и капсул, где она использовалась

## Работа с образами

### Создание образа

1. Перейдите в раздел "Образы" через меню навигации
2. Нажмите кнопку "Создать образ"
3. Заполните форму:
   - **Название** (обязательно) - название образа, например "Офисный образ"
   - **Описание** (опционально) - описание образа
   - **Вещи** - выберите вещи для образа
     - Можно выбрать несколько вещей
     - Порядок вещей можно изменить перетаскиванием
   - **Уровень формальности** (опционально) - от 1 (очень неформальный) до 5 (очень формальный)
   - **Сезон** (опционально) - выберите сезон
   - **Теги** (опционально) - выберите теги для образа
   - **Избранное** - отметьте, если хотите добавить образ в избранное
4. Нажмите "Сохранить образ"

### Просмотр образов

На странице "Образы" отображается список всех ваших образов. Каждая карточка показывает:
- Название образа
- Описание
- Вещи, входящие в образ (миниатюры)
- Уровень формальности
- Сезон
- Теги
- Статус избранного

### Фильтрация образов

Используйте фильтры для поиска нужных образов:
- По сезону
- По уровню формальности
- По избранным
- По тегам
- Поиск по названию

### Просмотр деталей образа

Кликните на карточку образа, чтобы открыть страницу с детальной информацией:
- Название и описание
- Все вещи образа с фотографиями
- Уровень формальности
- Сезон
- Теги
- Статус избранного
- Даты создания и обновления

### Редактирование образа

1. Откройте страницу деталей образа
2. Нажмите кнопку "Редактировать"
3. Измените нужные поля
4. Добавьте или удалите вещи
5. Нажмите "Сохранить изменения"

### Добавление вещи в образ

1. Откройте страницу деталей образа
2. Нажмите кнопку "Добавить вещь"
3. Выберите вещь из списка
4. Вещь будет добавлена в образ

### Удаление вещи из образа

1. Откройте страницу деталей образа
2. Найдите вещь, которую хотите удалить
3. Нажмите кнопку удаления рядом с вещью
4. Подтвердите удаление

### Избранные образы

Чтобы добавить образ в избранное:
1. Откройте страницу деталей образа
2. Нажмите кнопку "В избранное" (иконка звезды)

Чтобы убрать из избранного, нажмите кнопку еще раз.

### Удаление образа

1. На странице списка образов найдите нужный образ
2. Нажмите кнопку удаления
3. Подтвердите удаление

## Работа с капсулами

### Создание капсулы

Капсула - это группа вещей и образов для определенного периода или цели (например, "Весенняя капсула", "Рабочая капсула").

1. Перейдите в раздел "Капсулы" через меню навигации
2. Нажмите кнопку "Создать капсулу"
3. Заполните форму:
   - **Название** (обязательно) - название капсулы
   - **Описание** (опционально) - описание капсулы
   - **Сезон** (опционально) - выберите сезон
   - **Вещи** - выберите вещи для капсулы
   - **Образы** - выберите образы для капсулы (опционально)
4. Нажмите "Сохранить капсулу"

### Просмотр капсул

На странице "Капсулы" отображается список всех ваших капсул. Каждая карточка показывает:
- Название капсулы
- Описание
- Сезон
- Количество вещей
- Количество образов

### Просмотр деталей капсулы

Кликните на карточку капсулы, чтобы открыть страницу с детальной информацией:
- Название и описание
- Сезон
- Все вещи капсулы
- Все образы капсулы
- Даты создания и обновления

### Генерация образов из капсулы

Капсула позволяет автоматически генерировать образы из входящих в нее вещей:

1. Откройте страницу деталей капсулы
2. Нажмите кнопку "Сгенерировать образы"
3. Укажите количество образов для генерации (от 1 до 50)
4. Нажмите "Сгенерировать"
5. Просмотрите сгенерированные образы
6. Сохраните понравившиеся образы, указав название и другие параметры

**Примечание:** Генерация образов использует алгоритм комбинирования вещей из капсулы. Качество генерации зависит от количества и разнообразия вещей в капсуле.

### Просмотр комбинаций капсулы

1. Откройте страницу деталей капсулы
2. Нажмите кнопку "Комбинации"
3. Просмотрите все возможные комбинации вещей из капсулы

### Редактирование капсулы

1. Откройте страницу деталей капсулы
2. Нажмите кнопку "Редактировать"
3. Измените нужные поля
4. Добавьте или удалите вещи и образы
5. Нажмите "Сохранить изменения"

### Удаление капсулы

1. На странице списка капсул найдите нужную капсулу
2. Нажмите кнопку удаления
3. Подтвердите удаление

**Важно:** При удалении капсулы вещи и образы не удаляются, удаляется только сама капсула и связи с вещами/образами.

## Работа с тегами

### Просмотр тегов

1. Перейдите в раздел "Теги" через меню навигации
2. Просмотрите список всех доступных тегов:
   - **Системные теги** - предустановленные теги (нельзя удалить)
   - **Мои теги** - созданные вами теги

### Создание тега

1. Перейдите в раздел "Теги"
2. Нажмите кнопку "Создать тег"
3. Заполните форму:
   - **Название** (обязательно) - название тега
   - **Цвет** (опционально) - выберите цвет для тега
4. Нажмите "Сохранить"

**Примечание:** Теги можно также создавать прямо в формах создания/редактирования вещей и образов.

### Редактирование тега

1. Перейдите в раздел "Теги"
2. Найдите нужный тег
3. Нажмите кнопку редактирования
4. Измените название или цвет
5. Нажмите "Сохранить"

**Примечание:** Системные теги нельзя редактировать.

### Удаление тега

1. Перейдите в раздел "Теги"
2. Найдите нужный тег
3. Нажмите кнопку удаления
4. Подтвердите удаление

**Примечание:** Системные теги нельзя удалить.

### Использование тегов

Теги можно добавлять к вещам и образам:
- При создании/редактировании вещи или образа
- Через поиск тегов в форме
- Можно выбрать несколько тегов
- Можно создать новый тег прямо в форме

## Советы и рекомендации

### Организация гардероба

1. **Регулярно добавляйте вещи** - добавляйте вещи сразу после покупки
2. **Используйте теги** - теги помогают быстро находить вещи для определенных случаев
3. **Заполняйте все поля** - чем больше информации о вещи, тем проще ее найти и использовать
4. **Используйте заметки** - записывайте особенности вещи, размер, материал и т.д.

### Создание образов

1. **Экспериментируйте** - создавайте разные образы из одних и тех же вещей
2. **Используйте капсулы** - группируйте вещи по сезонам или стилям
3. **Генерируйте образы** - используйте автоматическую генерацию для поиска новых комбинаций
4. **Сохраняйте избранное** - отмечайте понравившиеся образы как избранные

### Использование капсул

1. **Планируйте заранее** - создавайте капсулы на сезон или для определенной цели
2. **Используйте генерацию** - автоматическая генерация образов помогает найти новые комбинации
3. **Обновляйте капсулы** - добавляйте новые вещи и удаляйте старые

### Аналитика

1. **Регулярно проверяйте аналитику** - это поможет понять, какие вещи вы используете чаще всего
2. **Используйте индекс использования** - найдите вещи, которые вы не используете, и подумайте, нужны ли они
3. **Анализируйте сочетаемость** - используйте карту сочетаемости для создания новых образов

## Часто задаваемые вопросы

### Как добавить несколько фотографий для одной вещи?

В текущей версии каждая вещь может иметь только одно изображение. При необходимости можно заменить изображение при редактировании вещи.

### Можно ли восстановить удаленную вещь?

Нет, удаление необратимо. Рекомендуется делать резервные копии базы данных.

### Как работает генерация образов?

Генерация образов использует алгоритм комбинирования вещей из капсулы. Система создает различные комбинации вещей, учитывая их категории и совместимость.

### Можно ли экспортировать данные?

В текущей версии экспорт данных не реализован. Эта функция может быть добавлена в будущих версиях.

### Как изменить пароль?

Перейдите в раздел "Профиль", нажмите "Редактировать" и введите новый пароль. Не забудьте указать текущий пароль для подтверждения.

### Можно ли использовать приложение на мобильном устройстве?

Приложение адаптировано для мобильных устройств и должно корректно работать на смартфонах и планшетах.

## Решение проблем

### Изображение не загружается

Проверьте:
- Формат файла (должен быть JPEG, PNG, GIF или WebP)
- Размер файла (не более 5MB)
- Наличие свободного места в базе данных

### Ошибка при сохранении

Проверьте:
- Все обязательные поля заполнены
- Данные соответствуют ограничениям (длина названия, размер файла и т.д.)
- Интернет-соединение стабильно

### Вещь не отображается

Проверьте:
- Фильтры не скрывают вещь
- Вещь принадлежит вашему аккаунту
- Попробуйте обновить страницу

### Проблемы с входом в систему

Проверьте:
- Правильность email и пароля
- Регистрация была завершена успешно
- Попробуйте очистить кеш браузера
