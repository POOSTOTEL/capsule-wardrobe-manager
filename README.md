# Капсульный Гардероб

Веб-приложение для создания и ведения капсульного гардероба.

## Технологический стек

- PHP 8.2
- PostgreSQL 15
- Nginx
- Docker & Docker Compose

## Быстрый старт

1. Клонируйте репозиторий
2. Скопируйте `.env.example` в `.env` и настройте переменные окружения:
   ```bash
   cp .env.example .env
